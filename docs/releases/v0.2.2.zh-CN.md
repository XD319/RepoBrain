# RepoBrain v0.2.2

这是针对 `v0.2.1` 之后暴露出的 Windows CI 格式校验问题的补丁版本。

## Fixed

- 新增仓库级 `.gitattributes`，确保 Windows CI 在 checkout `src/`、`test/` 等文本文件时统一使用 LF 行尾。
- 修复只在 `windows-latest` 上出现的远端 `Format check` 失败。

## Verification

- `npm run format:check`
- `git ls-files --eol src test`
- `npm run build`
- `npm run typecheck`
- `npm run lint`
- `npm test`
- `npm run smoke:package`

## Notes

- `v0.2.1` 仍然是 conversation-start 发布后的本地格式修复补丁版本。
- `v0.2.2` 是让同一套格式契约在 GitHub Actions Windows runner 上也稳定生效的后续补丁版本。
