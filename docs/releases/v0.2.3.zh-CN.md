# RepoBrain v0.2.3

这是在 `v0.2.2` 之后，用于简化维护者发布流程的补丁版本。

## Changed

- 新增 `npm run release:verify`，用一条命令跑完整发版门禁。
- 新增 `npm run release:publish`，底层由专用发布脚本驱动，可在 GitHub trusted publishing 与 `NPM_TOKEN` 回退之间自动选择。
- 更新发布 workflow 和维护者文档，让 GitHub Actions 默认走 trusted publishing，同时在迁移期继续支持 token 回退。

## Verification

- `npm run format:check`
- `npx vitest run test/publish-package-script.test.mjs test/package-manifest.test.mjs`
- `npm run release:verify`

## Notes

- `v0.2.3` 不改变终端用户看到的公开 CLI 契约。
- 这个补丁的重点是让 RepoBrain 的发版更容易执行，也更容易定位 npm 发布侧的配置问题。
