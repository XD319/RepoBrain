# Copilot Adapter

This directory contains a minimal RepoBrain custom instructions template for GitHub Copilot.

- Primary artifact: `copilot-instructions.md`
- Suggested destination: `.github/copilot-instructions.md`
- Durable memory source: `.brain/`

The template is intentionally thin so Copilot uses RepoBrain's shared knowledge model instead of a separate instruction-owned memory system.
