# Codex Adapter

This directory contains the minimal RepoBrain template for Codex.

- Primary artifact: `SKILL.md`
- Recommended inputs: `brain inject` output plus `brain suggest-skills` hints
- Durable memory source: `.brain/`

Copy `SKILL.md` into the Codex skill or instruction location you use for the repo, then tailor the workflow details while keeping RepoBrain as the shared memory core.
